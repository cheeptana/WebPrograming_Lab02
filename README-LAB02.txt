README (Lab02 10 Points)
--
- ข้อมูลการเชื่อมต่อฐานข้อมูล Private Key จะแจ้งให้ทราบในห้องสอบ
- โครงสร้าง JSON สำหรับจัดเก็บข้อมูลภายใต้คอลเลกชัน (Collections) Pets
{
    petId: String,
    petName: String,
    petBD: String,
    petType: Number,
    petNote: String,
    petOwner: String
}

กำหนดให้เพิ่มเติมการทำงานของแอปพลิเคชันให้ทำงานได้อย่างสมบูรณ์ ดังนี้
- เชื่อมโยงหน้าเพจในทุกหน้าให้สมบูรณ์
- กรอกข้อมูลในฟอร์มเพื่อดำเนินการบันทึก โดยระบุข้อมูลในหนึ่งรายการที่เพิ่มชื่อ-สกุลนักศึกษาในฟิลด์เจ้าของ เช่น Anna Devid xxxxxxxxxxxxxx-x ในคอลเลกชัน Pets อย่างน้อย 2 รายการ
- กรอกข้อมูลแก้ไขข้อมูลสัตว์เลี้ยง โดยระบุข้อมูลในหนึ่งรายการที่เพิ่มลงท้ายรหัสนักศึกษาในฟิลด์ชื่อสัตว์เลี้ยง เช่น HerbA xxxxxxxxxxxxxx-x ในคอลเลกชัน Pets เพียง 1 รายการ
- กำหนดให้ดึงข้อมูลสัตว์เลี้ยงในคอลเลกชัน Pets แสดงผลทุกเอกสาร (Documents)
- กำหนดให้เลือกลบรายการสัตว์เลี้ยงในคอลเลกชัน Pets ที่ไม่ต้องการได้
- กำหนดให้ดึงรายการสัตว์เลี้ยงในคอลเลกชัน Pets แสดงผลในแต่ละเอกสารเพื่อดูรายละเอียดได้

การเริ่มต้นทำงานแอปพลิเคชันฝั่ง Client
> npm run dev

การเริ่มต้นทำงานแอปพลิเคชันฝั่ง Server
> npm run server
